# Convert IFA

Convert IFADavies (2013) data to ISO country level.

## Usage

``` r
convertIFA(x)
```

## Arguments

- x:

  MAgPIE object containing IFA data region resolution

## Value

MAgPIE object of the IFA data disaggregated to country level

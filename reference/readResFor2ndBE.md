# readResFor2ndBE

Read in old ReMIND use of residues for 2nd generation bioenergy and
newly estimations

## Usage

``` r
readResFor2ndBE(subtype)
```

## Arguments

- subtype:

  oldReMIND, newAgriSupply

## Value

List of magpie objects with results on old ReMIND regions level

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

if (FALSE) { # \dontrun{

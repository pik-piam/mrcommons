# readSSPResults

Reads in a reporting mif file from the SSP scenario results

## Usage

``` r
readSSPResults()
```

## Value

MAgPIE object with regional aggregation of SSP regions, including all
indicators

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

## Author

Abhijeet Mishra, Florian Humpenoeder

``` r
if (FALSE) { # \dontrun{
readSource("SSPresults", aggregate = FALSE)
} # }
```

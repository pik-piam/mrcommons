# calcBioplasticToBiomass

calculates conversion factors from bioplastic demand to demand of
biomass needed for the production, taking into account the average share
of different biomass sources (glycerol, starch, sugars, cellulose, oils)
for bioplastic production and corresponding content in the different
crop types

## Usage

``` r
calcBioplasticToBiomass()
```

## Value

List of magpie objects with global conversion factors, unit and
description.

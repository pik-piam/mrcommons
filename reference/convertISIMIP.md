# convertISIMIP

convert data to ISO country level

## Usage

``` r
convertISIMIP(x, subtype)
```

## Arguments

- x:

  MAgPIE object on cellular level

- subtype:

  data subtype

## Value

MAgPIE object on country level

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

if (FALSE) { # \dontrun{

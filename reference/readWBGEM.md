# readWBGEM

read raw data of world commodity prices from the dataset of World Bank
global economic monitor

## Usage

``` r
readWBGEM()
```

## Value

magpie object of global price of commodities

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

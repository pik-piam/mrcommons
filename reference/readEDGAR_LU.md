# Read EDGAR_LU

Read-in EDGAR_LU csv file as magclass object

## Usage

``` r
readEDGAR_LU(subtype = "CO2")
```

## Arguments

- subtype:

  emission type

## Value

magpie object EDGAR_LU data

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

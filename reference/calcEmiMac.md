# Calculate baseline emissions for maccs

Provides REMIND data for baseline emissions for maccs.

## Usage

``` r
calcEmiMac()
```

## Value

REMIND data for baseline emissions for maccs and corresonding weights
(NULL) as a list of two MAgPIE objects

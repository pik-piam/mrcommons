# convertCMIP7_CEDS

converts harmonized emission data used for CMIP7, global data is removed
here, use `readSource(CMIP7_CEDS, convert = FALSE)` to obtain it.

## Usage

``` r
convertCMIP7_CEDS(x)
```

## Arguments

- x:

  magpie object from source function

## Value

MAgPIE object

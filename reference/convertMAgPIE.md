# Convert MAgPIE data

Convert MAgPIE data to country-level.

## Usage

``` r
convertMAgPIE(x, subtype)
```

## Arguments

- x:

  input MAgPIE data on region-level

- subtype:

  Either "EmiAirPoll", "macBase", "co2tax", "supplyCurve_magpie_40",
  "abatparam_co2"

## Value

magpie object

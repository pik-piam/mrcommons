# readFoodSystemsDashboard

read in Food Systems Dashboard indicators

## Usage

``` r
readFoodSystemsDashboard(subtype)
```

## Arguments

- subtype:

  Currently either "Processed" for Processed food expenditures per
  capita or "Industrial Processing Share"

## Value

governance index data at iso-country level

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

if (FALSE) { # \dontrun{

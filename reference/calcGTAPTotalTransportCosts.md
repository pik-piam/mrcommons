# calcGTAPTotalTransportCosts

Calculates iso-level total global transport costs from GTAP for GTAP
food commodities

## Usage

``` r
calcGTAPTotalTransportCosts(costType = "transport", version = "9")
```

## Arguments

- costType:

  transport or wholesale

- version:

  "81" or "9"

## Value

List of magpie object with results on country level, unit and
description.

# downloadNitrogenBoundariesGridded

Download he grid-level regional nitrogen boundary datasets from
Schulte-Uebbing et al. (2022)

## Usage

``` r
downloadNitrogenBoundariesGridded()
```

## See also

[`madrat::downloadSource()`](https://rdrr.io/pkg/madrat/man/downloadSource.html)

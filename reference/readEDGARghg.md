# readEDGAR_GHG

read EDGAR GHG emission data

## Usage

``` r
readEDGARghg(subtype = "ghg_by_sector")
```

## Arguments

- subtype:

  ghg_by_sector to read in the sheet GHG_by_sector_and_country, lulucf
  to read in the sheet LULUCF_countries

## Value

magpie objects of EDGAR historical emissions (MtCO2eq)

# downloadAndrijevic2019

This function downloads the governance indicator data available at
https://github.com/marina-andrijevic/governance2019

## Usage

``` r
downloadAndrijevic2019(subtype = "governance_obs_project")
```

## Arguments

- subtype:

  file that should be downloaded from governance2019 repository

# readHHS_USDA

reads calory requirement for a standardized population from HHS & USDA.
2015. "2015-2020 Dietary Guidelines for Americans." 8. Dietary
Guidelines for Americans.
https://health.gov/dietaryguidelines/2015/resources/2015-2020_Dietary_Guidelines.pdf.
Appendix 2

## Usage

``` r
readHHS_USDA()
```

## Value

Magpie object with results on global level.

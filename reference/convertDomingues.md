# convertDomingues

fills missing countries in Domingues data with NA, and transforms cost
units from 2018 US\$/GJ to 2017 US\$/GJ

## Usage

``` r
convertDomingues(x)
```

## Arguments

- x:

  unconverted magpie object from read-script

## Value

Data as MAgPIE object with common country list

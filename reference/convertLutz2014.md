# convertLutz2014

It fills the missing values of the output of readLutz2014 through the
weighted average of the values of two countries with similar
characteristics to the one that has na values.

## Usage

``` r
convertLutz2014(x)
```

## Arguments

- x:

  magpie object provided by the read function

# readBodirsky2018

Reads in regression parameters estimated using mrregression, and reads
in some scenario based on Bodirsky et al not yet published

## Usage

``` r
readBodirsky2018(subtype = "bmi_shr")
```

## Arguments

- subtype:

  bmi_share, demand_regression, intake_regression or
  bodyheight_regression for the estimated regression paramters of
  different regressions. scenarios for scenario projections.

## Value

magpie object

# Download GFED

Download the GFED (Global Fire Emissions Database) dataset for all years
not labeled as beta, in addition to the emission factors.

## Usage

``` r
downloadGFED()
```

## See also

[`madrat::downloadSource()`](https://rdrr.io/pkg/madrat/man/downloadSource.html)

# readCarbonLTS

Read-in an Long term carbon storage data for historical period.

## Usage

``` r
readCarbonLTS(subtype)
```

## Arguments

- subtype:

  data subtype.

## Value

magpie object of the FRA 2015 data

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

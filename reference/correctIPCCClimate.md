# correctIPCCClimate

Correct IPCC climate classification

## Usage

``` r
correctIPCCClimate(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

Magpie object with results on cellular level for 12 IPCC climate zone
types

``` r
if (FALSE) { # \dontrun{
```

# calcNitrogenFixationRateNatural

calculates fixation rates from natural ecosystems based on
evapostranspiration

## Usage

``` r
calcNitrogenFixationRateNatural(cells = "lpjcell")
```

## Arguments

- cells:

  "magpiecell" for 59199 cells or "lpjcell" for 67420 cells

## Value

List of magpie objects with results on global level, empty weight, unit
and description.

# calcLandArea

total land area in a cell

## Usage

``` r
calcLandArea(cells = "lpjcell")
```

## Arguments

- cells:

  "magpiecell" for 59199 cells or "lpjcell" for 67420 cells

## Value

magpie object in cellular resolution

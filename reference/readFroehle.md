# Read parameters of Froehle equations

Publication Froehle, Andrew W. 2008 "Climate Variables as Predictors of
Basal Metabolic Rate: New Equations" American Journal of Human Biology:
The Official Journal of the Human Biology Council 20 (5): 510-29. doi:
10.1002/ajhb.20769.

## Usage

``` r
readFroehle()
```

## Value

MAgPIE object

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

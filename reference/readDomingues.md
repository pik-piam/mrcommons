# readDomingues

Reads biomass yields (GJ/ha/year) and factor costs (US\$ 2018/GJ) for
bioenergy crops from Domingues et al., 2022 (DOI:
10.5281/zenodo.6913709)

## Usage

``` r
readDomingues()
```

## Value

Data as MAgPIE object with average yields and costs of betr and begr

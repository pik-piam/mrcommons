# Read in regression coefficients for central feed shares

Read in csv file containing coefficients of non-linear regression
analysis for the calculation of future central feed shares in feed
baskets dependent on livestock productivity trends

## Usage

``` r
readFeedShareReg()
```

## Value

MAgPIE object containing regression coefficients

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

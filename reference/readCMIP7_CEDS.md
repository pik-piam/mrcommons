# readCMIP7_CEDS

reads in harmonized emission data used for CMIP7 from three files.
country_file for energy related emissions on country level and global
bunker emissions, global_file for F-Gases and AFOLU on global level,
cmip7-historical-ghg-eq for GHG totals. Source:
https://zenodo.org/records/15059443

## Usage

``` r
readCMIP7_CEDS()
```

## Value

MAgPIE object

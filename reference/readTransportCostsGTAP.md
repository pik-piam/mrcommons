# Read Transport Costs

Read in Transport Costs from GTAP

## Usage

``` r
readTransportCostsGTAP()
```

## Value

Transport Costs in USD

## See also

[`madrat::readSource()`](https://rdrr.io/pkg/madrat/man/readSource.html)

## Author

David Chen

## Examples

``` r
if (FALSE) { # \dontrun{
a <- readSource(type = "TransportCostsGTAP")
} # }
```

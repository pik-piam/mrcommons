# historical emissions per sector or mac

Provides historical emissions values per economic sector or per mac
sector. For now it only includes European data.

## Usage

``` r
calcHistEmissions(subtype = "sector")
```

## Arguments

- subtype:

  Either "sector" or "MAC"

## Value

magpie object of historical emissions data

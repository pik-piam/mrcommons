# readLutz2014

It reads and clears the dataset of the global population projections by
age, sex and education, available on the Wittgenstein Centre Data
Explorer and published by Lutz, Butz and K. C. , 2014. "Population and
human capital in the twenty-first century" Oxford University Press. From
.csv file to a magclass object

## Usage

``` r
readLutz2014()
```

## Value

magpie object with the dataset downloaded. It contains missing values
and it is possible to replace them with the function convertLutz2014.

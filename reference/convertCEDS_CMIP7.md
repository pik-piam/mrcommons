# convertCEDS_CMIP7

converts harmonized emission data used for CMIP7, global data is removed
here, use `readSource(CEDS_CMIP7, convert = FALSE)` to obtain it.

## Usage

``` r
convertCEDS_CMIP7(x)
```

## Arguments

- x:

  magpie object from source function

## Value

MAgPIE object

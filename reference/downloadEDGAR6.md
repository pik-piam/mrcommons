# downloadEDGAR

download EDGAR emission data

## Usage

``` r
downloadEDGAR6(subtype = "n2o")
```

## Arguments

- subtype:

  type in gas for receiving the newest data. type in specific filename
  for old version 4.31 data

## Value

List of magpie objects with results on cellular level, weight, unit and
description.

# readIPCCClimate

Read IPCC climate classification

## Usage

``` r
readIPCCClimate()
```

## Value

Magpie object with results on cellular level for 12 IPCC climate zone
types

Kristine Karstens

``` r
if (FALSE) { # \dontrun{
```
